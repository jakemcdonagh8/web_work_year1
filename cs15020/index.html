Declaration of Originality

Due to the requirements of my assignment, I declare that the contents of this site consists of my work for the majority. Below this paragraph are the following resources that I used in order to assist me in certain aspects of this assignment:

http://www.w3schools.com/tags/default.asp: HTML Reference Sheet, used in order to find out certain attributes, how to use tables and lists, etc.

http://www.w3schools.com/cssref/default.asp: CSS Reference Sheet, used in order to find property colours/fonts. Was also used to find out how to center tables

http://www.w3schools.com/css/default.asp: CSS Tutorials; Basic tutorials of the features and capabilities of using a CSS. Highly useful as i've never attempted CSS before.

http://stackoverflow.com/questions/1513871/xhtml-wont-validate-and-in-a-javascript-function: Fix for why my XHTML wasn't validating correctly.

http://users.aber.ac.uk/ais/examples/css/asmt2/index.html: Angharad Shaw's example of a basic website with CSS. I used this as a template design for my site at the begining and slowly added more features onto it.
